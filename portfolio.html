BASIC WEB

Basic, you say? Despite calling it "basic", it was an intensive introduction to the primary tools of Multimedia Design. Human brains are hardwired to connect the dots and make sense out of everything the eyes see. In Basic Web I learned about different layout principles so we can dominate better the usage of positive space (represented by the elements we use) and negative space (the background). I was also introduced to a beautiful language: coding!


01.02.01 Basic mobile website

How we arrange the elements in the negative space we are given is crucial in order to create something that the brain can recognize easily.

The Gestalt Laws can be compared to the Ten Commendaments of the Design Bible. During this project I used the law of symmetry in the three column layout (for desktop) and single column layout (for mobile). I have also used the law of proximity in the navigation menu.

During Basic Mobile Website I discovered that html is like a canvas and css is our brush and paint.
